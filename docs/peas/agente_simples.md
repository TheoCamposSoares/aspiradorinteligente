# PEAS — Agente Simples

- **Medidas de desempenho:** Pontuação por quadrado limpo e penalização por locomoção.
- **Ambiente:** Duas salas que podem conter sujeiras ou não. 
- **Atuadores:** Mover para a esquerda, direita ou aspirar.
- **Sensores:** Sensor de localização, sensor de sujeira.- **Medidas de desempenho:** Alcançar o objetivo, evitar obstáculos, eficiência no caminho.
